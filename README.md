# Запуск сервера

Надо развернуть PostgreSQL и Redis

Создать в PostgresQL базу данных pumpkin

Затем надо оформить файл `.env`. Пример:

```
DB_PASSWORD=my_secure_password
DB_USER=tarasovxx
DB_PORT=5432

SERVER_PORT=8800

REDIS_PORT=6379
REDIS_USER=tarasovxx
REDIS_PASSWORD=my_secure_password
```
